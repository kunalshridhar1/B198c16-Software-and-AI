# AI-Powered Personalized Shopping Assistant

## Overview

The AI-Powered Personalized Shopping Assistant is an intelligent e-commerce support system designed to improve online shopping experiences through machine learning-driven recommendations, budget optimization, and deal discovery.

The application analyzes product information and user preferences to generate personalized recommendations and assist users in making informed purchasing decisions.

---

## Features

### AI Product Recommendation Engine

Uses TF-IDF Vectorization and Cosine Similarity to recommend similar products.

### Personalized Recommendations

Provides recommendations based on user-selected categories and interests.

### Budget Optimizer

Identifies highly-rated products that fit within a user's specified budget.

### Deal Discovery

Detects products with the highest discounts and savings opportunities.

### Smart Shopping Cart

Allows users to add products and calculate total spending.

### Purchase Reminders

Stores reminder information using SQLite.

### Analytics Dashboard

Provides visual insights into product categories and pricing.

---

## Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-Learn
* SQLite
* Matplotlib

---

## AI Techniques

### TF-IDF Vectorization

Converts textual product descriptions into numerical feature vectors.

### Cosine Similarity

Measures similarity between products to generate recommendations.
